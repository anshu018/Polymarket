# Phase 9 Plan 01: Hermes-Adapted Self-Learning & Memory System

## Objective
- **What:** Replicate the Hermes agent's self-learning, frozen snapshot memory system, and threat sanitization, surgically adapted for Polymarket.
- **Why:** Enable our 4 trading pipeline agents to learn from historical resolved trades, dynamically refining their accuracy, calibration, and category profiles over time.
- **Output:** A robust, Supabase-backed, threat-sanitized, and asynchronously reviewed memory layer (`learning/`) with pipeline hooks and unit tests.
