## Objective
- **What:** Configure the agent to run continuously on Hetzner CX22 with automated startup reconciliation, Telegram alerts, and auto-restart policies.

- **Output:** Continuous running main execution loop (`main.py`), reconciliation (`execution/reconciliation.py`), Telegram alerts (`monitoring/telegram_alerts.py`), and a `systemd` service configuration (`polymarket-agent.service`).
